# Frontend
Para apoyo al Código se especifican los siguientes requisitos:

1) Frontend
  - Trabajar en la rama de diseño.
  - Diseño de html, css con bootstrap botones, funcionales.

2) Backend
  - Trabajar en la rama código.
  - Diseño de interacciones con el usuario usando javascript.

3) Envio 
  - Enviar el push y posteriormente , se pasa a revisión de código y si todo esta bien : 'Aceptado'.
